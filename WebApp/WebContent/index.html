Project Description

For this project, I configured three virtual nodes based on Debian Squeeze (Debian 6.0). Each image is managed using VMWare Fusion.

  • Debian1 - Dedicted to Cassandra only
  • Debian2 - Dedicated to Analytics Hadoop/Hive
  • Debian3 - Dedicated to Solr/Enterprise Search

Data Model

The goal of the project is to create an e-commerce system for the food service industry that enables on-line consumers to purchase foods and restaurant supplies from a single source. The system will provide the consumer with basic point-of-sale functionality while enabling the provider to optimize their product offerings based on various product and consumer information.

Java Application - Initial Data Load

Most data files were loaded easily using the copy utility, but products required more processing. Each row required a loop to read and assign all of the values to the attributes collection. Another issue was encountered with vendor names. Some vendor names contain commas, which eliminated the use of the String.split() function. Instead, a CSV reader class was used to process the file.

tpstats/iostats

Java Application - Order Simulation

This process simulates an active web site by reading orders from the provided CSV and processing them as if the data was being submitted through a web site.

Each order object retrieves additional data about users, accounts, and products from C* and writes information to query tables.

For extra credit, the application also updates the balance on the Account table.
